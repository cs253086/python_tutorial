Lesson 07: Functions

organize game code with reusable functions

How to use this lesson

Use the right-side Python Playground panel to run code.

You can type tutorial snippets into the editor and click Run code.

Tutorial

Welcome to lesson 07.

In this lesson, you will learn how to:

  • create reusable functions with def
  • pass information using parameters
  • return values from functions
  • organize game code into small parts

At the end, your game code will be cleaner and easier to read.

What is a function?

A function is a named block of code.

You write it once, then call it many times.

Example:

def say_hello():
    print("Hello, explorer!")

say_hello()

1) Function with parameters

In the Playground editor, type:

def greet_player(name):
    print("Welcome,", name)

player_name = input("Name: ")
greet_player(player_name)

Click Run code. name is a parameter. It receives the value from player_name.

2) Function with return value

In the editor, type:

def add_score(current_score, points):
    new_score = current_score + points
    return new_score

score = 0
score = add_score(score, 10)
print("Score:", score)

Click Run code. return sends a value back.

3) Break game into small functions

In the editor, type:

def show_title():
    print("=== TREASURE HUNT ===")

def get_move():
    return input("Move (left/right): ")

def apply_move(move, score):
    if move == "left":
        print("Treasure found! +10")
        return score + 10
    elif move == "right":
        print("Empty room.")
        return score
    else:
        print("Invalid move.")
        return score

show_title()
score = 0
move = get_move()
score = apply_move(move, score)
print("Score:", score)

Now each part has one clear job.

Why functions help

  • less repeated code
  • easier to test one piece at a time
  • easier to read and fix

def show_title():
    print("=== TREASURE HUNT ===")


def get_move():
    return input("Move (left/right): ")


def apply_move(move, score):
    if move == "left":
        print("Treasure found! +10")
        return score + 10
    if move == "right":
        print("Empty room.")
        return score
    print("Invalid move.")
    return score


def main():
    show_title()
    score = 0
    move = get_move()
    score = apply_move(move, score)
    print("Score:", score)


if __name__ == "__main__":
    main()

Tutorial

Welcome to bonus lesson 10.

In this lesson, you will learn how to:

  • use Python's random module
  • pick random events
  • make your game different each time

Randomness makes replaying the game more fun.

1) Import random

In the Playground editor, type:

import random

number = random.randint(1, 6)
print("Dice roll:", number)

Click Run code. Run it a few more timesâ€”the number changes randomly.

2) Random choice from a list

In the editor, type:

import random

events = ["nothing", "trap", "treasure", "potion"]
event = random.choice(events)

print("Random event:", event)

Click Run code. The event changes each run.

3) Add random to game turn

In the editor, type:

import random

score = 0
lives = 3

for turn in range(1, 6):
    event = random.choice(["trap", "treasure", "empty"])
    print("Turn", turn, "- event:", event)

    if event == "trap":
        lives = lives - 1
        print("You lost 1 life.")
    elif event == "treasure":
        score = score + 10
        print("You gained 10 score.")
    else:
        print("Nothing happened.")

    print("Score:", score, "| Lives:", lives)

    if lives <= 0:
        print("Game over.")
        break

This version feels new each play.

4) Optional: random room order

You can shuffle a list:

import random

rooms = ["trap", "empty", "treasure", "empty"]
random.shuffle(rooms)
print(rooms)

shuffle changes list order randomly.

Solution code

import random


def main():
    score = 0
    lives = 3

    for turn in range(1, 6):
        event = random.choice(["trap", "treasure", "empty"])
        print("Turn", turn, "- event:", event)

        if event == "trap":
            lives = lives - 1
            print("You lost 1 life.")
        elif event == "treasure":
            score = score + 10
            print("You gained 10 score.")
        else:
            print("Nothing happened.")

        print("Score:", score, "| Lives:", lives)

        if lives <= 0:
            print("Game over.")
            break


if __name__ == "__main__":
    main()
